Default Fonts

Default fonts, also known as system fonts, are the pre-installed typefaces on most computers and devices.
They are designed to be highly legible and work consistently across different platforms.

Using the default typeface and removing custom typefaces and the font-family declaration in the website code can help improve a website's environmental impact. Eliminating custom typefaces will make a web page load faster due to fewer external requests to the server. This not only reduces the energy used to load the website but also helps to aid a positive user experience. The absence of custom typefaces and logos can create a cleaner and more minimalist design, simplifying the user pathway and appealing to users who prefer simplicity and clarity. Our website 'Streamlined' is an example of a website coded without the font-family declaration and is all default text.
